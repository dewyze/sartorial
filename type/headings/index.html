Headings

Sartorial uses a ratio based system for determining heading sizes

Headers

The Sartorial Type modules provides the following header tags.

Html tag Widescreen size Desktop size Tablet size Phone size
h1 48.8px 48.8px 39.8px 28.8px
h2 39.1px 39.1px 33.2px 25.6px
h3 31.3px 31.3px 27.6px 22.8px
h4 25.0px 25.0px 23.0px 20.3px
h5 20.0px 20.0px 19.2px 18.0px
h6 16.0px 16.0px 16.0px 16.0px

The default ratios between the texts are:

  • Widescreen: $ratio-major-third (1.250)
  • Desktop: $ratio-major-third (1.250)
  • Tablet: $ratio-minor-third (1.200)
  • Phone: $ratio-major-second (1.125)

Font & Line Height Variables

The type sizes are built off of a base font-size and a ratio. Additionally, there are separate variables that can be used to build those up per media size. See the table below for what you can set and the defaults.

Variable Default Value Details
$font-size 1.6rem Default font-size in rem as 10% of pixel size
$font-sans-serif Roboto Default sans-serif font, used with `.serif`
$font-serif Georgia Default serif font, used with `.sans-serif`
$line-height 1.5 Default line-height
$line-height-phone $line-height Phone matches default line height
$line-height-tablet 1.4 Default line-height for tablets
$line-height-desktop 1.4 Default line-height for desktops
$line-height-widescreen 1.4 Default line-height for widescreens

Available Ratios

The ratio between the main height and the header size is based on an explicit ratio. You can either use one of these or your own. The ratios are set per media size:

Variable Default (See below table for details) Details
$ratio-global None Ratio for all media sizes. Not recommended as a default as type size should scale appropriately for different devices.
$ratio-phone Major Second Ratio for $screen-phone breakpoint
$ratio-tablet Minor Third Ratio for $screen-tablet breakpoint
$ratio-desktop Major Third Ratio for $screen-desktop breakpoint
$ratio-widescreen Major Third Ratio for $screen-widescreen breakpoint

You can set your own ratio or use one of the below variables.

Variable Step Value Ratio Name
$ratio-minor-second Minor Second 1.067
$ratio-major-second Major Second 1.125
$ratio-minor-third Minor Third 1.2
$ratio-major-third Major Third 1.25
$ratio-perfect-fourth Perfect Fourth 1.333
$ratio-augmented-fourth Augmented Fourth 1.414
$ratio-perfect-fifth Perfect Fifth 1.5
$ratio-golden-ratio Golden Ratio 1.618
