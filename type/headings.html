Headings

Sartorial uses a ratio based system for determining heading sizes

Usage

{% include snippet.html file="type/h.html" %}

Why

Type is important. It can be hard to find good settings for headers that are also responsive. Sartorial sets a ratio that deterines the relationship between each font-size instead of needing to manually define it.

I decided to not require any classes for text and just add the styling to the heading tags directly to ease use.
