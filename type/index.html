Typography

Sartorial's Type module provides classes, mixins, and defaults for creating a responsive, scaled, and decorated typography system

Usage

{% include snippet.html file="type/h.html" %}

Customize

The type sizes are built off of a base font-size and a ratio. Additionally, there are separate variables that can be used to build those up per media seize. See the table below for what you can set and the defaults.

Variable Default Value
$font-size 1.6rem
$font-sans-serif 'Roboto'
$font-serif Georgia
$line-height 1.5
$line-height-phone $line-height
$line-height-tablet 1.4
$line-height-desktop 1.6
$line-height-widescreen 1.6

Available Ratios

Ratio Name Step Value
Minor Second 1.067
Major Second 1.125
Minor Third 1.200
Major Third 1.250
Perfect Fourth 1.333
Augmented Fourth 1.414
Perfect Fifth 1.500
Golden Ratio 1.618

Why

Ratios

The header and paragraph styling is designed to respect a ratio between the font-size (<p> tags) and the various headers. <h6> will match the font-size of the <p> tags, and each header after that will be 1 step higher.

The ratios are also, by default, different for different screen sizes.

Screen Size Ratio
Phone Major Second (1.067)
Tablet Minor Third (1.200)
Desktop and up Major Third (1.250)
