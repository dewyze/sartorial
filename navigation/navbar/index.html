Navbar

Sartorial provides classes for common navbar styles

Headers

The Sartorial Nav module provides the following classes for building responsive navbars:

Html tag Widescreen size Desktop size Tablet size Phone size
h1 48.8px 48.8px 39.8px 28.8px
h2 39.1px 39.1px 33.2px 25.6px
h3 31.3px 31.3px 27.6px 22.8px
h4 25.0px 25.0px 23.0px 20.3px
h5 20.0px 20.0px 19.2px 18.0px
h6 16.0px 16.0px 16.0px 16.0px

The default ratios between the texts are:

  • Widescreen: $ratio-major-third (1.250)
  • Desktop: $ratio-major-third (1.250)
  • Tablet: $ratio-minor-third (1.200)
  • Phone: $ratio-major-second (1.125)

Why

Type is important. It can be hard to find good settings for headers that are also responsive. Sartorial sets a ratio that deterines the relationship between each font-size instead of needing to manually define it.

I decided to not require any classes for text and just add the styling to the heading tags directly to ease use.

Customize

Navbar variables

The type sizes are built off of a base font-size and a ratio. Additionally, there are separate variables that can be used to build those up per media size. See the table below for what you can set and the defaults.

Variable Default Value Details
$navbar-height 3.25rem Default height of navbar
$navbar-spacing 1.5rem Default spacing between navbar elements
$color-navbar-background #ffffff (white) Background color of navbar
$color-navbar-background-hover #fafafa (light gray) Background color of navbar element on hover
