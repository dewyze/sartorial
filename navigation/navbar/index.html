Navbar

Sartorial provides classes for common navbar styles

There are two types of navbars available in sartorial: Split; and Centered. Both styles of navbar will collapse on mobile devices.

Split Navbar

If you want to make a collapsible navbar with the logo, left side, and right side, look no further!

    • .navbar - class that includes all navbar elements
      • .navbar-item-logo - this only differs from the other element types in that the background color does not change (normally or on hover)
        • .navbar-image - this is used in .navbar-item- to set the size of the image and pad it correctly to maintain $navbar-height
      • input.navbar-item- - any items that appear before the navbar toggle will not be collapsed on mobile
      • label.navbar-toggle - this forms the hamburger menu that can be clicked on mobile
        • - these are needed to form the hamburger menu visible on mobile
      • .navbar-item - this is an item on the left side of the navbar (to the right of the logo)
      • .navbar-item-right - this is an item on the right side of the navbar

Centered navbar

If you need a centered section of a navbar

    • .navbar - class that includes all navbar elements
      • .navbar-left - class for what goes on the left side of the navbar
        • .navbar-item - regular old navbar-item
      • .navbar-middle - class for what goes in the middle of the navbar
        • .navbar-brand - class for your logo
        • .navbar-item - regular old navbar-item
      • .navbar-right - class for what goes on the right side of the navbar
        • .navbar-item - regular old navbar-item

The .navbar element does not need a surrounding container. It will respect the $container-width variable and center the nav.

.container elements contain padding by default. If you had a .container.no-pad element around a nav, it would function the same way.

Customize

Navbar variables

The type sizes are built off of a base font-size and a ratio. Additionally, there are separate variables that can be used to build those up per media size. See the table below for what you can set and the defaults.

Variable Default Value Details
$navbar-height 3.25rem Default height of navbar
$navbar-spacing 1.5rem Default spacing between navbar elements
$navbar-vertical-padding 2/3 of $navbar-spacing Default padding above and below a navbar element
$color-navbar-background #ffffff (white) Background color of navbar
$color-navbar-background-hover #fafafa (light gray) Background color of navbar element on hover
